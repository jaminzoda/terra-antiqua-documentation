--- layout: tool title: Change Map Symbology working: true --- Change Map Symbology

This tool can be used to change the color palette of a map. The color schemes you may use in this tool are taken from the folder described here. Terra Antiqua is delivered with the color palettes described here.

Step-by-step

  1. Click on the icon . The Standard Processing window will pop up.

  2. In the field How would you like to process the input DEM, choose Change Map Symbology.

  3. In Raster to be modified, choose the layer you want to change the symbology of.

  4. Under the list Color palette, select the color scheme you want to apply to the map.

  5. Choose the Output file path, and click Run.
