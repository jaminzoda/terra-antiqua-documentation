Version 1.0

This tool can be used to change the color palette of a map. Terra Antiqua is delivered with the color palettes described here. You may choose one of those or add another one.

Step-by-step

  1. Click on the icon . The Standard Processing window will pop up.

  2. In the field How would you like to process the input DEM, choose Change Map Symbology.

  3. In Raster to be modified, choose the layer you want to change the symbology of.

  4. Under the list Color palette, select the color scheme you want to apply to the map.

  5. Go to Add custom color palette to add another color scheme.

    Please remember that the color palettes must be in .cpt format. The name of the color palette that will be listed in Terra Antiqua should be added in the first line of the .cpt file, following a hashtag.

  6. Choose the Output file path, and click Run.
