The reconstruction framework is set using the primary tools, while the following secondary tools are needed for essential tasks at various stages of the completion of the reconstruction.

The secondary tools include the following eight algorithms:

  • Remove Artefacts

  • Prepare Masks


These six tools are contained in the module called Standard processing tools:

  • Fill Gaps

  • Copy/Paste Raster

  • Isostatic Compensation

  • Smooth Raster

  • Set New Sea Level

  • Calculate Bathymetry

  • Change Map Symbology
